# atmacup18

# ディレクトリ構造
```
|- data
|   |- original_data   # 元データ
|   |- created_data    # 自分で処理や変換によって生成されたデータ
|   └─ public_data     # 公開されているデータ
|
|- notebooks           # ノートブックファイル（Jupyterなど）
|
|- outputs             # 出力ファイル（結果やレポートなど）
|
└─ src (this)*         # ソースコード（メインコード、スクリプトなど）
```


# 動作
```sh
docker compose up
```
